La sintassi Base di Java

Impara la sintassi base di Java e le variabili base.

Informazioni principali

  • Java è case-sensitive

    int treeCount = 15;

    è vista come una variabile diversa da

    int TreeCount = 25;

  • Nomi delle classi e nome del file

    In Java, ogni classe deve avere la prima lettera maiuscola. Inoltre, ogni file deve avere il nome della sua classe pubblica, per esempio:

    public class FirstProgram

    Andrà salvato in un file chiamato FirstProgram.java
