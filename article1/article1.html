La sintassi Base di Java

Impara la sintassi base di Java e le variabili base.

Informazioni principali

  • Java è case-sensitive

    int treeCount = 15;

    è vista come una variabile diversa da

    int TreeCount = 25;

  • Nomi delle classi e nome del file

    In Java, ogni classe deve avere la prima lettera maiuscola. Inoltre, ogni file deve avere il nome della sua classe pubblica, per esempio:

    public class FirstProgram

    Andrà salvato in un file chiamato FirstProgram.java

  • Primo programma in Java - Hello World

    Vediamo come si scrive, in Java, il classico programma "Hello World"

    public class FirstProgram { public static void main (String[] args) { System.out.println("Hello World"); } }

    Diamo un'occhiata al codice, il main() è la funzione base, chiamata anche "entry-point" del programma, ovvero il punto dal quale il programma inizierà ad eseguire tutte le nostre istruzioni.
    In seguito troviamo l'istruzione di output di default, evidenziata da "System.out.println" che stamperà nel terminale una stringa da noi fornita prima di andare a capo.
